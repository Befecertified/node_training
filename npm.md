NPM - Node Package Manager

Some basic commands and meaning

npm install <package name> - to install a package
npm i <package name> - short form on the install
npm i <package name>@x.x.x - install a particular version of a package
npm i -g <package name> - install a package globally
npm uninstall <package name> - to uninstall a package
npm un <package name> - short form to uninstall a package
npm i <package name> --save-dev - to install package as developer dependency
npm outdated - to check outdated packages
npm update - to update outdated packages
npm -v - to check npm version
npm init - to initialization a node project
npm init --yes - automatically initialize a repo
